**분석 방법**:
- 각 miRNA가 카테고리별로 관련된 경로 수(Npath) 계산
- 가중치를 적용하여 다중 카테고리 커버리지 우보 종점으로 계산점함

**최종 표의 "다수 커버" 선정 근거**:
최종 표의 다중 커버리지 점수 = (inflam × 0.25) + (fib × 0.25) + (anti × 0.20) + (endo × 0.20) + (cvd × 0.10) + (sen × 0.05)

## 3단계: 다중 커버 후보 압축 및 Top-N 선별 (Step-3 → Step-4)

### Step-3: 다중 커버 후보로 (재압축 리스트)

**선별 알고리즘**:
```
점수 = (다중 카테고리 커버리지) + (FC 가중) + (MT_mean 검출량)

우선순위:
1. 다중 카테고리 균형 커버 (6개 카테고리 골고루 분포)
2. 높은 발현 변화 (FC > 30)
3. 충분한 발현량 (MT_mean 검출 양호)
```

**Top-N(Primary/Backup) 압축**:
- Primary: 최우선 후보
- Backup: 예비 후보

| **Step-1** | 후보 99종(1차 후) | 검출 및 상향 조절 조건을 만족하는 후보를 확정하고, 실험 실행 가능 규모로 1차 압축함 | "고우선 후보" 단축 리스트 |
| **Step-2** | 고우선 후보군(압축 리스트) | CKD-CVD 핵심 병태 측(염증/항산화/섬유화/내피/심혈관/노화)를 카테고리별 경로 커버리지(Npath) 산출하고, 다수 커버 우보 중점으로 계산점함 | 최종 표의 "다수 커버" 선정 근거 |
| **Step-3** | 다중 커버 후보로(재압축 리스트) | (다중 카테고리 커버리지) + (FC) + (MT_mean) 통합하여 후보를 랭킹화고 Top-N(Primary/Backup) 압축함 | 1차/2차 후보군 도출 근거 |
| **Step-4** | Top-N 후보군 | MSC 라인 수율 최소화하면서 CKD-CVD 핵심 축을 동시에 커버하는 **Core-2(2종)** 최종 확정함 | 최종 표의 Core-2: hsa-miR-4739, hsa-miR-4651 |
| **Step-5** | Core-2 및 칵테일 후보 | 효능 판정을 위해 족별 **1차 판독지표(Primary readouts)** 와 Go/No-Go 판정값을 확정함 | 최종 표의 Primary readouts + Go/No-Go 문장 |
| **Step-6** | Core-2 기반 칵테일 설계 | 제조·비교의 재현성을 위해 post-isolation 후합, particles 기준 1:1을 기본으로 고정하고, 필요 시 copy/particle로 비율을 보정함 | 최종 표/분문의 칵테일 설계 근거(1:1) |

## 💡 핵심 선별 근거 (Why Core-2?)

### 1️⃣ **과학적 근거**
```
┌─────────────────────────────────────────────────────────────┐
│ Core-2가 99개 중 선택된 이유                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ ✅ 다중 카테고리 균형 커버                                    │
│    - miR-4739: 6개 카테고리 모두 골고루 분포 (균형형)         │
│    - miR-4651: 염증·내피·CVD 집중 커버 (특화형)              │
│    → 조합 시 상호 보완으로 전 카테고리 효율적 커버             │
│                                                             │
│ ✅ 높은 발현 변화 및 검출량                                   │
│    - miR-4739: FC=33.1 (상위 25%)                          │
│    - miR-4651: FC=109.5 (상위 6%, 매우 높음)                │
│    → 생체 내 치료 효과 기대                                  │
│                                                             │
│ ✅ 최대 경로 커버리지                                        │
│    - miR-4739: 62개 경로 (전체 3위)                         │
│    - miR-4651: 58개 경로 (전체 1위)                         │
│    → CKD-CVD 복합 병태의 다중 표적 동시 조절                  │
│                                                             │
│ ✅ 명확한 검증 지표                                          │
│    - 각 miRNA별 Primary readouts 설정 완료                  │
│    - Go/No-Go 기준 명확화                                   │
│    → 체계적 효능 평가 가능                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2️⃣ **실용적 근거**
```
┌─────────────────────────────────────────────────────────────┐
│ 최소 구성 (2종) 전략의 장점                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 🔬 제조 단순화                                               │
│    - 2종 miRNA만 엑소좀에 로딩                               │
│    - 비율 제어 용이 (1:1 기본, 최적화 가능)                   │
│    - 품질관리 간소화                                         │
│                                                             │
│ 💊 임상 적용성                                               │
│    - 복잡도 감소 → 규제 승인 용이                            │
│    - 재현성 향상 → 대량 생산 가능                            │
│    - 안전성 평가 효율화                                      │
│                                                             │
│ 📈 확장 가능성                                               │
│    - Core-2 검증 후 Core-3, Core-4로 확장 가능               │
│    - 환자 맞춤형 조합 개발 기반                              │
│    - Backup 후보 준비 완료 (검증 실패 시 대체)                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3️⃣ **임상적 근거**
```
┌─────────────────────────────────────────────────────────────┐
│ CKD-CVD 통합 치료 전략                                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 🏥 CKD 진행 단계별 대응                                       │
│    - 초기(G1-G2): 염증·산화 스트레스 조절 (miR-4651/4739)    │
│    - 중기(G3-G4): 섬유화 억제 집중 (miR-4739)                │
│    - 말기(G5): 심혈관 합병증 예방 (miR-4651)                  │
│                                                             │
│ ❤️ CVD 합병증 동시 예방                                       │
│    - CKD 환자의 주요 사망 원인 = CVD                          │
│    - Core-2는 신장+혈관+심장 모두 타깃                        │
│    - 통합 치료로 예후 개선 극대화                             │
│                                                             │
│ 🎯 정밀의료 기반                                             │
│    - 환자별 병태에 따라 miRNA 비율 조절 가능                  │
│    - Primary readouts로 개인별 반응 모니터링                 │
│    - 데이터 기반 치료 최적화                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📌 모식도 시각화 권장 요소

이 문서를 바탕으로 시각화 모식도를 제작할 때 포함하면 좋을 요소:

### 상단: 전체 프로세스 플로우
- Step-0부터 Step-6까지 화살표로 연결
- 각 단계마다 입력(99개→리스트→Core-2→엑소좀) 명시
- 색상 코딩: 각 단계별 다른 색상

### 중단: 선별 기준 및 근거
- 6개 카테고리 원형 차트 (가중치 비율 표시)
- Core-2 miRNA 프로파일 카드 (FC, 경로 수, 강점 표시)
- Primary readouts 아이콘 (염증/산화/섬유화/내피)

### 하단: 타깃 엑소좀 구조
- 엑소좀 단면도 (표면 타깃팅 펩타이드 + 내부 miRNA)
- 질환 부위 타깃팅 화살표 (신장/혈관/심장)
- 치료 효과 아이콘 (각 조직별)

### 디자인 요소
- 색상: 의학적 blue/green 계열
- 아이콘: 신장, 심장, 혈관, 세포, 엑소좀
- 폰트: 깔끔한 한글 sans-serif
- 레이아웃: 좌→우 또는 상→하 흐름
