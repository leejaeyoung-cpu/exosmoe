# CKD-CVD 작용 시그널 경로 상세 분석

## 1. CKD (만성 신장질환) 주요 시그널 경로

### 1.1 염증 시그널 (Inflammatory Signaling)
**병리 기전:**
- IL-17 경로 활성화 → Th17 세포 매개 염증
- NF-κB 과활성 → TNFα, IL-6 과다 생산
- JAK-STAT 경로 활성 → 사이토카인 신호 증폭
- NOD-like receptor 활성 → 선천면역 과활성

**Core-2 miRNA 작용:**
- **miR-4651** → NF-κB 억제
- **miR-4739** → IL-17, JAK-STAT 경로 조절
- 결과: 염증성 사이토카인 폭풍 차단

### 1.2 섬유화 시그널 (Fibrotic Signaling)
**병리 기전:**
- TGF-β → Smad2/3 활성화 → 섬유화 유전자 발현
- ECM (콜라겐, 피브로넥틴) 과다 축적
- EMT (상피간엽이행) → 신장 세포의 섬유아세포 전환
- Wnt/β-catenin 활성 → 섬유화 진행

**Core-2 miRNA 작용:**
- **miR-4739** → TGF-β/Smad 경로 차단
- 결과: COL1A1, αSMA 발현 감소, 신장 기능 보존

### 1.3 산화 스트레스 시그널 (Oxidative Stress)
**병리 기전:**
- 미토콘드리아 기능 장애 → ROS 과다 생산
- Glutathione 고갈 → 항산화 능력 저하
- Ferroptosis (철 의존성 세포사) 활성

**Core-2 miRNA 작용:**
- **miR-4739** → Mitophagy 활성화 (손상 미토콘드리아 제거)
- **miR-4739** → FoxO 경로 활성화 (항산화 유전자 발현)
- 결과: ROS 감소, ΔΨm (막전위) 회복

## 2. CVD (심혈관질환) 주요 시그널 경로

### 2.1 내피 기능장애 시그널 (Endothelial Dysfunction)
**병리 기전:**
- VCAM1/ICAM1 과발현 → 염증세포 부착 증가
- Focal adhesion 약화 → 세포-기질 연결 손상
- Fluid shear stress 반응 저하 → 혈류 적응 실패
- HIF-1 경로 이상 → 저산소 반응 장애

**Core-2 miRNA 작용:**
- **miR-4651** → VCAM1/ICAM1 정상화
- **miR-4739** → Focal adhesion 강화, HIF-1 조절
- 결과: 내피 기능 회복, 혈관 반응성 개선

### 2.2 심근병증 시그널 (Cardiomyopathy)
**병리 기전:**
- AGE-RAGE 경로 활성 → 당뇨병성 심장 손상
- Adrenergic signaling 이상 → 심근 수축력 저하
- Cardiac muscle contraction 장애
- Dilated/Hypertrophic cardiomyopathy 진행

**Core-2 miRNA 작용:**
- **miR-4739** → AGE-RAGE 경로 차단
- **miR-4739** → Adrenergic signaling 조절
- 결과: 심근 보호, 수축력 개선

### 2.3 동맥경화 시그널 (Atherosclerosis)
**병리 기전:**
- Fluid shear stress → 내피 손상 → 플라크 형성
- Platelet activation → 혈전 형성
- 염증성 대식세포 침윤

**Core-2 miRNA 작용:**
- **miR-4651** → 염증 억제로 대식세포 침윤 차단
- **miR-4739** → Platelet activation 조절
- 결과: 동맥경화 진행 억제

## 3. CKD-CVD 연결 시그널 (Cross-talk)

### 3.1 CKD → CVD 악화 경로
1. **염증 → 내피 손상**: CKD의 만성 염증이 혈관 내피를 직접 손상
2. **산화 스트레스 → 동맥경화**: ROS가 LDL 산화 촉진 → 플라크 형성
3. **요독 물질 → 심근 독성**: CKD로 축적된 요독 물질이 심장 직접 손상

### 3.2 CVD → CKD 악화 경로
1. **혈류 감소 → 신장 허혈**: 심부전으로 신장 혈류 저하
2. **내피 기능장애 → 사구체 손상**: 전신 내피 장애가 신장 모세혈관 포함
3. **고혈압 → 사구체 경화**: 혈압 상승이 신장 여과 기능 손상

### 3.3 Core-2 칵테일의 통합 치료 전략
**miR-4651 (염증/내피 특화):**
- CKD 염증 → CVD 내피 손상 연결고리 차단
- NF-κB 억제 → VCAM1/ICAM1 정상화

**miR-4739 (다중 표적):**
- CKD 섬유화 차단
- CVD 심근 보호
- 공통: 산화 스트레스 감소

**시너지 효과:**
- 염증 + 섬유화 동시 차단 → CKD 진행 원천 봉쇄
- 내피 + 심근 보호 → CVD 합병증 예방
- 산화 스트레스 감소 → CKD-CVD 악순환 끊기

## 4. 시간대별 시그널 변화 예측

### 단기 (1-3개월)
**CKD 시그널:**
- ↓ NF-κB 활성 → ↓ TNFα, IL-6
- ↓ ROS → ↑ 미토콘드리아 기능

**CVD 시그널:**
- ↓ VCAM1/ICAM1 → ↓ 염증세포 부착
- ↑ 내피 반응성

### 중기 (3-6개월)
**CKD 시그널:**
- ↓ TGF-β/Smad → ↓ COL1A1, αSMA
- ↓ 섬유화 진행 → 신기능 안정화

**CVD 시그널:**
- ↑ 혈관확장 반응
- ↓ 동맥경화 진행

### 장기 (6-12개월)
**통합 효과:**
- CKD 단계 진행 둔화
- CVD 이벤트 (심근경색, 뇌졸중) 위험 감소
- 생존율 개선

---

## 5. 검증 지표 (Primary Readouts)

### CKD 지표
- **염증**: TNFα, IL-6, p-p65
- **섬유화**: COL1A1, αSMA, p-Smad2/3
- **산화**: ROS, ΔΨm
- **신기능**: 혈청 크레아티닌, eGFR, 단백뇨

### CVD 지표
- **내피**: VCAM1, ICAM1, tube formation
- **심근**: 심박출량, Adrenergic 반응
- **혈관**: 혈압, 내피 의존성 혈관확장
- **동맥경화**: IMT (내중막 두께), 플라크 크기

---

**결론:**
Core-2 miRNA 칵테일(miR-4739 + miR-4651)은 CKD와 CVD의 핵심 시그널 경로를 동시에 차단하여, 두 질환의 악순환을 끊고 통합 치료 효과를 제공합니다.
