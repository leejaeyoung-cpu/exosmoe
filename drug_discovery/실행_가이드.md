# CKD-CVD AI 신약 발견 - 실행 가이드

## 🚀 빠른 시작

### 방법 1: 실행파일(.exe) 사용 (권장)

#### Step 1: 실행파일 빌드

1. `빌드_실행파일.bat` **더블클릭**
2. 자동으로 PyInstaller 설치 및 빌드 진행 (3-5분)
3. 완료되면 `dist` 폴더에 `CKD_CVD_DrugDiscovery.exe` 생성

#### Step 2: 실행

1. `dist` 폴더로 이동
2. `CKD_CVD_DrugDiscovery.exe` **더블클릭**
3. 파이프라인 자동 실행
4. 결과는 `results` 폴더에 저장

**⚠️ Windows Defender 경고**
- "Windows의 PC 보호" 창이 뜨면
- "추가 정보" 클릭 → "실행" 클릭

---

### 방법 2: Python 직접 실행

#### Python이 설치된 경우

1. `실행.bat` **더블클릭**
   
   또는 터미널에서:
   ```bash
   cd drug_discovery
   python run_pipeline.py
   ```

## 🛠️ 문제 해결

### PyInstaller 설치 오류

```bash
# 수동 설치
pip install pyinstaller
```

### "Python이 인식되지 않습니다"

- Python을 PATH에 추가해야 합니다
- Python 재설치 시 "Add to PATH" 체크

### 실행 시 DLL 오류

```bash
# Visual C++ Redistributable 필요
# Microsoft 사이트에서 다운로드
```

### 빌드가 너무 오래 걸림

- 정상입니다 (3-5분)
- PyTorch 등 큰 라이브러리 포함
